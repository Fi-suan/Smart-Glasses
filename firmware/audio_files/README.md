# Аудио файлы для DFPlayer Mini

## Структура SD карты

Создайте на microSD карте (FAT32) следующие папки и файлы:

```
/
├── 01/                         # Системные звуки
│   ├── 001.mp3                # Подключение к телефону
│   ├── 002.mp3                # Отключение
│   ├── 003.mp3                # Ошибка
│   ├── 004.mp3                # Успех
│   └── 005.mp3                # Низкий заряд батареи
│
├── 02/                         # Навигация
│   ├── 001.mp3                # "Поверните налево"
│   ├── 002.mp3                # "Поверните направо"
│   ├── 003.mp3                # "Двигайтесь прямо"
│   ├── 004.mp3                # "Вы прибыли к месту назначения"
│   ├── 005.mp3                # "Развернитесь"
│   ├── 006.mp3                # "Через 50 метров"
│   ├── 007.mp3                # "Через 100 метров"
│   └── 008.mp3                # "Маршрут перестроен"
│
├── 03/                         # Предупреждения
│   ├── 001.mp3                # "Внимание, впереди препятствие"
│   ├── 002.mp3                # "Осторожно, ступенька"
│   ├── 003.mp3                # "Впереди лестница"
│   ├── 004.mp3                # "Осторожно, дорога"
│   └── 005.mp3                # "Пешеходный переход"
│
└── 04/                         # Уведомления
    ├── 001.mp3                # "Новое сообщение"
    ├── 002.mp3                # "Входящий вызов"
    └── 003.mp3                # "Напоминание"
```

## Требования к файлам

- **Формат**: MP3 (рекомендуется 128-192 kbps)
- **Частота**: 44100 Hz или 22050 Hz
- **Каналы**: Mono или Stereo
- **Громкость**: Нормализуйте все файлы до одного уровня

## Как создать аудио файлы

### Вариант 1: Онлайн TTS (рекомендуется)

Используйте онлайн сервисы text-to-speech:
- https://cloud.google.com/text-to-speech (Google)
- https://speechgen.io/ (русский язык)
- https://ttsmaker.com/

### Вариант 2: Windows встроенный TTS

PowerShell скрипт для генерации:
```powershell
Add-Type -AssemblyName System.Speech
$synth = New-Object System.Speech.Synthesis.SpeechSynthesizer
$synth.SetOutputToWaveFile("output.wav")
$synth.Speak("Поверните налево")
$synth.SetOutputToDefaultAudioDevice()
```

### Вариант 3: Запись голоса

Запишите фразы сами используя любой диктофон или программу.

## Проверка

После копирования файлов на SD карту:

1. Вставьте карту в DFPlayer
2. Подключите питание
3. Отправьте команду с телефона: `{"cmd": "play", "data": {"track": 1}}`
4. Должен воспроизвестись файл 001.mp3 из корня или первой папки

## Формат карты

**ВАЖНО**: Карта должна быть отформатирована в FAT32!

Windows:
1. Правый клик на диске → Форматировать
2. Файловая система: FAT32
3. Размер единицы распределения: По умолчанию
4. Быстрое форматирование: Да
